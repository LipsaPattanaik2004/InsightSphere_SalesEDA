# PROJECT OVERVIEW
__________________________
InsightSphere_SalesEDA is an end-to-end sales data analytics project built to explore patterns, trends, and revenue flow using structured datasets. It includes data cleaning, feature extraction, EDA, visualizations, and actionable business insights suitable for analytics and reporting roles.

## FEATURES INCLUDED
__________________________
Preprocessing & validation of sales data
Time-series revenue trend analysis
Region-wise & product-wise performance breakdown
Feature engineering (Revenue, Month, Year extraction)
Insight-rich visualizations (bar, line, distribution charts)
Export of cleaned dataset for BI dashboards (Power BI, Tableau)

### TECH STACK
__________________________
Python, Pandas, NumPy
Matplotlib / Seaborn
jupyter Notebook
SQL for data extraction workflows

#### PROJECT STRUCTURE
__________________________
InsightSphere_SalesEDA/
│── sales_eda.ipynb
│── README.md
└── data/
    └── sales_data.csv

##### HOW TO USE
__________________________
Clone or download the repository
Place your dataset inside the data/ folder
Open sales_eda.ipynb in Jupyter
Run the cells sequentially to generate EDA results & insights
